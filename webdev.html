HTML

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

CSS

Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML.

JavaScript

JavaScript, often abbreviated as JS, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS.

SQL

Structured Query Language, abbreviated as SQL, is a domain-specific language used in programming and designed for managing data held in a relational database management system.

Ruby

Ruby is a dynamic, open source programming language with a focus on simplicity and productivity.

React

React is a free and open-source front-end JavaScript library for building user interfaces based on UI components.

PHP

PHP is a general-purpose scripting language geared toward web development.

Django

Django is a free and open-source, Python-based web framework that follows the model–template–views architectural pattern.

Angular

Angular is a platform for building mobile and desktop web applications.

Express Js

Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.

MongoDB

The developer data platform that provides the services and tools necessary to build distributed applications fast, at the performance and scale users demand.

My SQL

MySQL is an open-source relational database management system.

Next-JS

Next.js is an open-source web development framework created by Vercel enabling React-based web applications with server-side rendering and generating static websites.

Node-JS

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine.

Postgresql

PostgreSQL also known as Postgres, is a free and open-source relational database management system (RDBMS) emphasizing extensibility and SQL compliance.

Vue-JS

Vue.js is an open-source model–view–viewmodel front end JavaScript framework for building user interfaces and single-page applications.
